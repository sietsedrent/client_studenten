Welkom!

Deze pagina gebruikt Vite als voorbeeld voor het samenvoegen(bundling) en minimaliseren (minify) van de code.

  • npm -install: gebruikt NPM om de dependencies te installeren.
  • npm run build: gebruikt NPM om het "vite build" commando uit te voeren en een "dist" map te genereren
  • in de vite.config.js zie je instellingen inclusief "minify: true"
